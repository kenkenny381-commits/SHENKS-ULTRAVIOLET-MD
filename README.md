# Shenks Ultraviolet MD
A WhatsApp bot built with Baileys.

## Features
- Anti commands
- AI chat
- Media download
- Group management
- And more

## Deploy
### Heroku
[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy)

### Manual
1. Clone repo
2. `npm install`
3. Add `SESSION_ID` to `.env`
4. `npm start`

## Requirements
- Node.js 18+
- FFmpeg
- ImageMagick
